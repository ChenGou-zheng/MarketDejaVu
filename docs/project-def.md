Project 3 中国股市历史行情与证券交易策略匹配研究


问题描述：自1992年至今，中国股市经历了多种行情。我们可以利用机器学习的方法，对历史行情进行学习与分类。例如，以2008年11月至2009年5月这段行情为例，可以通过当时发布的宏观政策、产业政策，以及大盘走势K线图、波动率、最大回撤、收益率等多个因子，为这段行情打出一个综合标签。依此类推，可以为不同历史行情段分别打上标签。这样一来，当未来某些特征再次出现时，当前行情走势很可能与历史上的某段行情高度相似。如果在历史那段行情中，某种策略表现良好，或某些行业、个股涨幅突出，那么当相似行情再次重演时，大概率这些策略或行业、个股仍会表现较好。此时，可以直接采用该策略，以期获取收益。

当然，这种方法并不能保证每次预测都能盈利。但如果一年预测10次，其中有6次判断正确，胜率即为60%，相当于6次盈利、4次亏损，总体来看仍然可以实现正收益。

考虑到1992年与当前市场环境差异较大，建议可仅选取2014年至今、约12年的历史行情进行学习。当然，如果同学们希望追溯更长时间的数据，也完全可以。

关于策略数据：目前可能没有覆盖那么长时间的策略数据，请同学们先思考解决方案。实际操作时，策略数据可从2020年开始获取。因此，我们可以用2020年至2025年的数据，对这5年的行情进行机器学习与分段标签标注。分段的长度不固定，例如2020年1月至3月，具体几个月应依据行情特征持续的时间来决定。然后，观察2026年出现的行情是否与2020—2025年间的某段行情相似。如果相似，就调用该段行情中表现良好的策略，检验其在2026年相似行情阶段是否依然业绩出色。若效果良好，则证明该方法有效。

===

主要任务:
1. 机器学习对历史行情进行学习与分类，打上综合标签。
   1. 主要的参数包括: 当时发布的宏观政策、产业政策，以及大盘走势K线图(技术面)、波动率、最大回撤、收益率等多个因子。
2. 判断当前行情走势和历史行情走势相似度
3. 判断在历史行情中哪一些策略表现良好
4. 根据当前行情走势调用历史行情表现良好策略的方式选择, 以及辅助决策的分析和参考数据.

===

- 技术面：看价格、K线、成交量、指标，研究“市场怎么走”。
- 基本面：看财报、估值、行业前景、护城河，研究“公司值不值”。
- 资金面：看成交量、北向资金、主力净流入、融资融券余额，研究“钱在往哪去”。你之前提到的大单净量就属于这一面。资金流向, 板块轮动.
- 消息面/政策面/情绪面：看宏观政策、产业新闻、突发事件、市场情绪，研究“什么在影响预期”。赚钱效应影响资金面.
